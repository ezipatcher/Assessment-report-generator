# 📊 AI Assessment Report Generator

An AI-powered Assessment Report Generator that evaluates candidate responses, calculates performance scores, generates professional summaries, and provides hiring recommendations using Google's Gemini AI.

## ✨ Features

- 📋 Multiple assessment templates
- 🤖 AI-generated performance summaries
- 📈 Automatic overall score calculation
- ✅ Recommendation engine (Proceed / Consider / Reject)
- ⚠️ Incomplete assessment handling
- 📊 Live score visualization
- 📄 ATS-compatible JSON export
- 📱 Responsive dashboard
- 🔒 Server-side AI processing with fallback support

## 🛠 Tech Stack

- Vite
- TypeScript
- React
- Tailwind CSS
- Express.js
- Google Gemini API
